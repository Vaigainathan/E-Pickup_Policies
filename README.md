# EPickup Policies

Static GitHub Pages site serving privacy policies for EPickup apps.

## URLs (after Pages is enabled on `main`, root)
- Landing: `https://<user>.github.io/epickup-policies/`
- Customer Privacy: `https://<user>.github.io/epickup-policies/customer/`
- Driver Privacy: `https://<user>.github.io/epickup-policies/driver/`
- Admin (future): `https://<user>.github.io/epickup-policies/admin/` (placeholder)

## Structure
- `index.html` — landing page linking to each app
- `customer/index.html` — Customer app privacy policy
- `driver/index.html` — Driver app privacy policy
- `assets/style.css` — shared minimal styles

## Update steps
1) Edit the relevant HTML page with new policy text.
2) Commit and push to `main`.
3) GitHub Pages will update automatically.

If you prefer source markdown, keep `.md` files in `policies/` and paste updates into the HTML pages.
